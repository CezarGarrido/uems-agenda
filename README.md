# uems-agenda

## Instalação

```
$ git clone https://github.com/CezarGarrido/uems-agenda.git
```

## Preparando o ambiente

Crie um novo ambiente virtual:
```
$ python -m venv venv
```

Ative o ambiente:

No Linux ou Mac:
```
$ . venv/bin/activate
```

No Windows:

```
> venv\Scripts\activate
```

Instale as dependências

```
$ pip install -r requirements.txt
```
